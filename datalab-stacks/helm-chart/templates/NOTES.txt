Use the following URL to access your Jupyter notebook:
{{/* Check if there is a secret already deployed (e.g. user used helm install --wait) */}}
{{- $secret := (lookup "v1" "Secret" .Release.Namespace (include "datasciencelab.fullname" .)) }}
{{- if $secret }}
  http://localhost/lab?token={{ index $secret.data "jupyterToken" | b64dec }}
{{- else}}
    http://localhost/lab?token=


You need to append Jupyter's secret token to the URL above. On Linux, use the following command:

    kubectl get secret --namespace {{ .Release.Namespace }} {{ include "datasciencelab.fullname" . }} -o jsonpath={.data.jupyterToken} | base64 --decode

On Windows, use the following command and then decrypt the base64-encoded token using e.g. www.base64decode.org

    kubectl get secret --namespace {{ .Release.Namespace }} {{ include "datasciencelab.fullname" . }} -o jsonpath={.data.jupyterToken}
{{- end}}
